Intelligence artificielle

Architecture multi-agents · détection comportementale · vision par ordinateur

Modèles & Vision
YOLOv8 (Ultralytics) — Détection d'objets en temps réel frame par frame
DeepSort — Tracking multi-objets (suivi des personnes entre les frames, ID persistant)
OpenCV — Extraction frames, annotations visuelles (rectangles, bannières colorées)
Pose estimation — Analyse de la posture pour détection de chute (ratio hauteur/largeur bounding box)
YOLOv8 DeepSort OpenCV 4.8 NumPy
Architecture Multi-Agents
Agent Chute — Détecte les chutes (pose + vitesse + seuil de confiance)
Agent Attroupement — Compte les personnes (seuil CROWD_MIN=8, configurable)
Agent Objet Abandonné — Objet sans propriétaire (grace period, suivi de présence)
Agent Décision — Agrège les résultats → niveau de risque : low / medium / high
4 agents spécialisés Analyse vidéo différée Streaming live

Sécurité Informatique

7 mécanismes de sécurité · couverture OWASP Top 10

Mécanismes implémentés
1 · JWT HS256 — Authentification (clé 64 hex, exp 24h, header uniquement)
2 · RBAC — Autorisation par rôles (admin / user, @token_required, @admin_required)
3 · Flask-Limiter — Anti brute-force (10/min, 50/h par IP → HTTP 429)
4 · bcrypt — Hash mots de passe (cost 12, salt aléatoire, jamais en clair)
Mécanismes implémentés (suite)
5 · Upload sécurisé — Whitelist 7 formats (secure_filename, max 500MB)
6 · CORS + Docker network — Isolation réseau (origines explicites, bridge privé)
7 · Audit Log — Traçabilité (activity_log MongoDB : IP, user, action, timestamp UTC)
+ Security Headers — CSP, X-Frame, X-XSS + Protection SSRF sur URLs caméras
Couverture OWASP Top 10
