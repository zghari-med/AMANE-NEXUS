🏗️ ARCHITECTURE SYSTÈME
═══════════════════════════════════════════════════════════════════════════

PIPELINE MULTI-AGENTS:

Vidéo Input
    │
    ▼
┌─ AGENT PERCEPTION (YOLOv8n) ─┐
│ Détection objets/personnes   │
│ Bounding boxes + confidence  │
└─────────┬─────────────────────┘
          │
          │ Redis: channel:detections
          │
          ▼
┌─ AGENT TRACKING (DeepSORT) ──┐
│ Appairage détections         │
│ IDs persistants              │
│ Calcul vitesse               │
└─────────┬─────────────────────┘
          │
          │ Redis: channel:tracks
          │
          ▼
┌─ AGENT ANALYSE ──────────────┐
│ Détection chutes             │
│ Détection attroupements      │
│ Détection objets abandonnés  │
└─────────┬─────────────────────┘
          │
          │ Redis: channel:analysis
          │
          ▼
┌─ AGENT DÉCISION ─────────────┐
│ Évaluation des risques       │
│ Génération alertes           │
│ Escalade de risque           │
└─────────┬─────────────────────┘
          │
          │ MongoDB + Notifications
          │
          ▼
       UI/Dashboard


COMMUNICATION INTER-AGENTS:
- Redis Pub/Sub (temps réel)
- 4 canaux: detections → tracks → analysis → alerts
- Throttling pour optimiser la CPU


STOCKAGE PERSISTANT:
- MongoDB pour: Users, Videos, Analyses, Alerts
- Indexes optimisés pour les requêtes fréquentes

PRÉREQUIS:
✅ Python 3.10+
✅ Node.js 18+
✅ MongoDB 6+
✅ Redis 6+
✅ 2GB+ RAM
✅ 10GB+ Disque


DÉMARRAGE RAPIDE:
1. Lancer MongoDB (mongod)
2. Lancer Redis (redis-server)
3. Lancer les 4 Agents Python
4. Lancer l'API Flask
5. Lancer le Frontend React
6. Ouvrir http://localhost:3000


PORTS:
- Frontend: 3000 (Vite dev server)
- Backend API: 5000 (Flask)
- MongoDB: 27017
- Redis: 6379
